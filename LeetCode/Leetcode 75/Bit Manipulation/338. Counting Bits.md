## My solution

```python
class Solution:
    def countBits(self, n: int) -> List[int]:
        # bits(2N) = bits(N)
        # bits(2N + 1) = 1 + bits(N)

        res = [0]
        for i in range(1, n + 1) :
            res.append(res[i >> 1] + i % 2)
        
        return res
```
